# JettyAI Tasks

This repository contains two different tasks related to web scraping and content analysis.

## Project Structure

- **[task1/](task1/)**: Multi-Source Web Scraper & Trust Scorer (Initial Version).
  - Uses `scrappers/` package.
  - Features AI-powered tagging and trust scoring.
- **[task2/](task2/)**: Medical Content Scraper & Trust Scoring System (Improved/Modular Version).
  - Uses `scraper/`, `scoring/`, and `utils/` packages.
  - Includes content chunking and advanced trust scoring algorithms.

## Getting Started

Each task is self-contained. Navigate to the respective directory and follow the instructions in the `README.md` file provided there.

### Typical Usage

```bash
cd task1
python main.py

# OR

cd task2
python main.py
```
